# iCloudEMS V2

A mobile-first web application that replicates the iCloudEMS V2 report card experience with:

- a dashboard of frequently used actions
- a report card form with semester/session selection
- biometric authentication flow for attendance scanning
- notifications and result views

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Capacitor
- Lucide React
- Recharts

## Features

- Mobile-style UI layout optimized for a phone frame
- Dashboard shortcuts for common student actions
- Report card form with submit flow and results view
- Biometric authentication modal before attendance scanning
- Bottom navigation for quick screen switching

## Getting Started

### Prerequisites

- Node.js
- npm

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

## Notes

- The app uses a mobile-first layout and is intended to run well in a browser or through Capacitor.
- Some assets and data are currently hardcoded for demo purposes.
